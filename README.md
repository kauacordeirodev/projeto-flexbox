# 📦 Projeto Flexbox: Justify Content

Este projeto é uma ferramenta interativa e educativa desenvolvida para visualizar, em tempo real, o comportamento da propriedade **`justify-content`** do CSS Flexbox. Através de uma interface intuitiva, você pode alternar entre diferentes valores de alinhamento e observar instantaneamente como os elementos se distribuem dentro de um container.

O objetivo principal é servir como um laboratório prático para desenvolvedores que desejam dominar o posicionamento de eixos no desenvolvimento web.

Tudo é executado diretamente no navegador utilizando **HTML5**, **CSS3** e **JavaScript**!

---

## 🚀 Funcionalidades

* **Visualização em Tempo Real:** Alteração imediata do layout ao clicar nos botões de controle.
* **Interface Limpa e Moderna:** Container estilizado com suporte a transições suaves para uma melhor experiência visual.
* **Domínio do Eixo Principal:** Demonstração prática dos 6 principais valores de distribuição:
    * `flex-start`
    * `flex-end`
    * `center`
    * `space-evenly`
    * `space-around`
    * `space-between`
* **Feedback Interativo:** Botões com efeitos de *hover* e escala para facilitar a navegação.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do container e dos controles.
* **CSS3:** Estilização avançada, uso de **Flexbox** para o layout e animações de transição (`transition`).
* **JavaScript:** Manipulação direta do **DOM** (*Document Object Model*) para alteração dinâmica de estilos inline.
