<h1 align="left">📦 Projeto Flexbox: Justify Content</h1>

###

<p align="left">Este projeto é uma ferramenta interativa e educativa desenvolvida para visualizar, em tempo real, o comportamento da propriedade justify-content do CSS Flexbox. Através de uma interface intuitiva, você pode alternar entre diferentes valores de alinhamento e observar instantaneamente como os elementos se distribuem dentro de um container.<br><br>O objetivo principal é servir como um laboratório prático para desenvolvedores que desejam dominar o posicionamento de eixos no desenvolvimento web — afinal, alinhar coisas no CSS não precisa ser um mistério.<br><br>Tudo é executado diretamente no navegador utilizando HTML5, CSS3 e JavaScript!</p>

###

<h2 align="left">🚀 Funcionalidades</h2>

###

<p align="left"><b>- Visualização em Tempo Real:</b> Alteração imediata do layout ao clicar nos botões de controle.<br><br>
<b>- Interface Limpa e Moderna:</b> Container estilizado com suporte a transições suaves para uma melhor experiência visual.<br><br>
<b>- Domínio do Eixo Principal:</b> Demonstração prática dos 6 principais valores de distribuição:<br><br>`flex-start`<br><br>`flex-end`<br><br>`center`<br><br>`space-evenly`<br><br>`space-around`<br><br>`space-between`<br><br><b>- Feedback Interativo:</b> Botões com efeitos de hover e escala para facilitar a navegação.</p>

###

<h2 align="left">🛠️ Tecnologias Utilizadas</h2>

###

<p align="left"><b>HTML5:</b> Estruturação semântica do container e dos controles.<br><br><b>CSS3:</b> Estilização, uso de Flexbox para o layout e animações de transição (transition).<br><br><b>JavaScript:</b> Manipulação direta do DOM para alteração dinâmica de estilos inline.</p>
